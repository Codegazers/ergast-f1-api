Pit stops API


If at the bottom of the url you add these parameters you can change the limit and offset values, by default they are set to 30 and 0
?limit=100&offset=2


/pitstops/year/round

E.G. /pitstops/2019/2

Returns the pit stops made in the second race of 2019


/pitstops/year/round/pitstop/numberPitStop

E.G. /pitstops/2019/2/pitstop/1

Returns to the pit stops made in the second race of 2019 only the first ones


/pitstops/year/round/pitstop/numberPitStop/driver/driverId

E.G. /pitstops/2019/2/pitstop/1/driver/stroll

Returns the first pitstop made in the second race of 2019 by the selected driver


/pitstops/year/round/driver/driverId

E.G. /pitstops/2019/2/driver/vettel

Returns the pit stops made the second race of 2019 by the selected driver


/pitstops/year/round/lap/numberLap

E.G. /pitstops/2019/2/lap/1

Returns the pit stops made on the first lap in the second race of 2019


/pitstops/year/round/lap/numberLap/driver/driverId

E.G. /pitstops/2019/2/lap/1/driver/stroll

The pitstop performed on the first lap of the second race in 2019 by the selected driver returns
