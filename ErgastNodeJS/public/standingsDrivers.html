Standings Drivers API


If at the bottom of the url you add these parameters you can change the limit and offset values, by default they are set to 30 and 0
?limit=100&offset=2


/standings/drivers/year

E.G. /standings/drivers/2019

Returns the driver rankings of the selected year

/standings/drivers/year/driver/driverId

E.G. /standings/drivers/2019/driver/leclerc

Returns the selected pilot's position in the chosen year

/standings/drivers/year/round

E.G. /standings/drivers/2018/2

Returns the driver rankings of the selected year after the selected round

/standings/drivers/year/round/driver/driverId

E.G. /standings/drivers/2018/2/driver/leclerc

Returns the selected pilot's position in the year chosen after the selected round

/standings/drivers/current

E.G. /standings/drivers/current

Return the current year's driver rankings

/standings/drivers/current/driver/driverId

E.G. /standings/drivers/current/driver/leclerc

Returns the position of the pilot selected in the current year

/standings/drivers/current/round

E.G. /standings/drivers/current/2

Return the current year's driver rankings after the selected round

/standings/drivers/current/round/driver/driverId

E.G. /standings/drivers/current/2/driver/leclerc

Returns the position of the pilot selected in the current year after the selected round

/standings/drivers/position/numberPosition

E.G. /standings/drivers/position/1

Returns all the drivers who arrived at the end of the championship in the chosen position

/standings/drivers/position/numberPosition/driver/driverId

E.G. /standings/drivers/position/1/driver/leclerc

Returns all the times when the selected driver has reached the end of the championship in the chosen position

/standings/drivers/driverId

E.G. /standings/drivers/vettel

Returns the end position of the selected driver
