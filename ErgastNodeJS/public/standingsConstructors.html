Standings Constructors API


If at the bottom of the url you add these parameters you can change the limit and offset values, by default they are set to 30 and 0
?limit=100&offset=2


/standings/constructors/year

E.G. /standings/constructors/2019

Returns the constructors ranking of the selected year

/standings/constructors/year/constructor/constructorId

E.G. /standings/constructors/2019/constructor/ferrari

Returns the position of the selected constructor in the chosen year

/standings/constructors/year/round

E.G. /standings/constructors/2018/2

Returns the constructors' ranking of the year selected after the selected round

/standings/constructors/year/round/constructor/constructorId

E.G. /standings/constructors/2018/2/constructor/ferrari

Returns the position of the selected constructor in the year chosen after the selected round

/standings/constructors/current

E.G. /standings/constructors/current

Returns the constructors ranking of the current year

/standings/constructors/current/constructor/constructorId

E.G. /standings/constructors/current/constructor/ferrari

Returns the position of the selected constructor in the current year

/standings/constructors/current/round

E.G. /standings/constructors/current/2

Returns the constructors ranking of the current year after the selected round

/standings/constructors/current/round/constructor/constructorId

E.G. /standings/constructors/current/2/constructor/ferrari

Returns the position of the selected constructor in the current year after the selected round

/standings/constructors/position/numberPosition

E.G. /standings/constructors/position/1

Returns all the constructors who arrived at the end of the championship in the chosen position

/standings/constructors/position/numberPosition/constructor/constructorId

E.G. /standings/constructors/position/1/constructor/ferrari

Returns all the times the selected constructor has reached the end of the championship in the chosen position

/standings/constructors/constructorId

E.G. /standings/constructors/ferrari

Return the end position of the selected constructor
