Constructors API


If at the bottom of the url you add these parameters you can change the limit and offset values, by default they are set to 30 and 0
?limit=100&offset=2

/constructors/all

E.G. /constructors/all

Returns all constructors

/constructors/name/constructorsId

E.G. /constructors/name/ferrari

Returns the selected constructor information

/constructors/year

E.G. /constructors/2019

Returns all the constructors who participated in the Grand Prix in the selected year

/constructors/year/round

E.G. /constructors/2019/2

Returns all the constructors who participated in the Grand Prix indicated in the selected year

/constructor/constructorStandings/position

/constructor/constructorStandings/1

Returns all the builders who arrived at a particular position at the end of the championship

/year/2019/circuit/circuitId/constructor/constructorId/driver/driverId/grid/position/result/position/fastest/rank/status/statusId

You can create a custom query using these parameters. NB you don't need to use them all
