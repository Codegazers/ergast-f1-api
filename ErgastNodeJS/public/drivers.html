Drivers API


If at the bottom of the url you add these parameters you can change the limit and offset values, by default they are set to 30 and 0
?limit=100&offset=2

/drivers/all

E.G. /drivers/all

Returns all drivers

/drivers/year

E.G. /drivers/2019

Returns all the drivers who participated in the Grand Prix in the selected year

/drivers/year/round

E.G. /drivers/2019/2

Returns all the drivers who participated in the Grand Prix indicated in the selected year

/drivers/name

E.G. /drivers/vettel

Returns the selected driver information
