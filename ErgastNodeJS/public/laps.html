Laps API


If at the bottom of the url you add these parameters you can change the limit and offset values, by default they are set to 30 and 0
?limit=100&offset=2


/laps/year/round/driver/driverId

E.G. /laps/2019/2/driver/hamilton

Returns the lap times made the second race of 2019 by the selected driver


/laps/year/round/lap/numberLap/driver/driverId

E.G. /laps/2019/2/lap/1/driver/hamilton

Returns the lap time of the second race of 2019 by the selected driver


/laps/year/round/lap/numberLap

E.G. /laps/2019/2/lap/1

Returns the lap times the second race of 2019 made in the selected lap


/laps/year/round

E.G. /laps/2019/2

Returns the lap times made in the second race of 2019 for all drivers
