Race Schedule API


If at the bottom of the url you add these parameters you can change the limit and offset values, by default they are set to 30 and 0
?limit=100&offset=2


/races/year

E.G. /races/2019

Returns all the races of the selected year

/races/year/round

E.G. /races/2019/2

Returns the race selected from the round of the selected year

/races/current

E.G. /races/current

Returns all the races of the current year

/races/current/round

E.G. /races/current/2

Returns the race selected from the current year round

/year/2019/round/2/circuit/circuitId/constructor/constructorId/driver/driverId/grid/position/result/position/fastest/rank/status/statusId

You can create a custom query using these parameters. N.B. you don't need to use them all
